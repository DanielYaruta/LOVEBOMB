# LOVEBOMB 💣

Учебный проект по JavaScript: таймер обратного отсчёта, оформленный как "бомба любви" — плюшевый мишка держит устройство с экраном, на котором идёт отсчёт времени.

🔗 **Демо:** [danielyaruta.github.io/LOVEBOMB](https://danielyaruta.github.io/LOVEBOMB/)

## Что умеет

- **Выбор времени** от 1 до 10 минут
- **Обратный отсчёт**, встроенный прямо в экран бомбы на изображении
- Каждую минуту под таймером **на 2.5 секунды появляется сообщение** (вместо стандартного `alert()`)
- По окончании отсчёта:
  - таймер показывает «BOOM!»
  - **200 сердец** разлетаются из экрана бомбы во все стороны
  - **100 сердец** падают сверху по всей картинке
- Кнопка **«Сброс»** возвращает таймер в исходное состояние без перезагрузки страницы
